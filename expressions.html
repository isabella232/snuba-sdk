class snuba_sdk.expressions.Column(name)[source]¶

Bases: snuba_sdk.expressions.Expression

A representation of a single column in the database. Columns are expected to be alpha-numeric, with ‘.’, ‘, and `: allowed as well. If the column is subscriptable then you can specify the column in the form column[key]. The column attribute will contain the outer column and key will contain the inner key.

Parameters

name (str) – The column name.

Raises

InvalidExpression – If the column name is not a string or has an invalid format.
